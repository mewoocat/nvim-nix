My personal NeoVim configuration packages as a nix flake.

### Overview
The key properties of this nvim config
- nix for plugin package management
- A focus on using lua for config

### Build
`nix build .#`

### Run
Locally:   `nix run .#`
Remotely:  `nix run github:mewoocat/nvim-nix`
